# Project Space Race

A data analysis school project exploring real SpaceX launch data, launch sites, success rates, and landing predictions.

## Overview
This was a school assignment focused on practicing data collection, cleaning, database management, analysis, and visualization using SpaceX launch records.

## What It Includes
- Web scraping SpaceX data
- Data wrangling and cleaning
- SQLite database creation and SQL queries
- Launch site and orbit analysis
- Interactive maps and visualizations (Plotly + Matplotlib)
- Machine learning model for landing success prediction

## Project Structure
- `Code/` — All Python scripts (web scraper, analysis, maps, predictions)
- `Data/` — CSV files and processed datasets
- `Notebooks/` — Jupyter notebooks for exploration and SQL labs
- `my_data1.db` — The main SQLite database

## Technologies Used
Python • Pandas • SQLite • Plotly • Matplotlib • Web Scraping
